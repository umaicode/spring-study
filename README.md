## 📝 작성 규칙 (Conventions)
- 각 강의 폴더에는 반드시 `README.md`를 둡니다.
- 강의에서 배우는 내용 (정리/노트/트러블슈팅)은 **강의 폴더 내부 README**에 기록합니다.
- 루트 `README.md`는 저장소 목적/구조/강의 인덱스만 관리합니다.
- OS/IDE 자동 생성 파일은 커밋하지 않습니다. (예: `.DS_Store`, `.idea/`, `build/` 등)
